# notas-atletas
Título: Aplicação de Processamento de Dados de Atletas (DevStart BAHIA)

Desafio: Implementar uma aplicação completa de cálculo de médias, utilizando o nome e as notas de avaliação de múltiplos atletas como input.

Execução: O projeto, desenvolvido como requisito final do curso DevStart BAHIA, exigiu a criação de uma interface (ou rotina) robusta para coletar dados de entrada, realizar o tratamento e configuração das notas, executar o cálculo algorítmico da média e, por fim, exibir o output consolidado (nome, notas e média) para cada registro de atleta.

Habilidades Demonstradas: Manipulação de inputs, lógica de cálculo, estrutura de dados (para armazenar múltiplos atletas) e output formatado.
